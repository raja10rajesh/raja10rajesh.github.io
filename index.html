Customised hardware acclerator for AI applications using FPGA

Genralised AI acclerator with custom MMU, Cache, Memory controller. Based on RISC-V

Combining the reconfigurable flexibility of the FPGA with the RISC-V ISA. A custom FP32-ALU along with a dedicated matrix multiplication units were designed. Sequential memory accessing is equiped with spatial R/W buffers for virtual cache emulation.
Result: Reconfigurable flexible genralised AI acclerator.

  • Year: 2020
  • Type: Research, Ongoing
